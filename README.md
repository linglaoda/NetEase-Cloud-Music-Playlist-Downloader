# NetEase-Cloud-Music-Playlist-Downloader

#### 本脚本基于 Python3 及 [NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi) 项目
## 本项目多数功能有赖于 NeteaseCloudMusicApi 项目，非常感谢 NeteaseCloudMusicApi 项目开发者的辛勤开发

## ✨项目特点
#### ▫ 可以一次性下载歌单内的所有歌曲
#### ▫ 下载歌曲默认最高音质(会员登陆后最高支持Hi-Res)
#### ▫ 下载格式为 mp3 或 flac 并非 NCM 无播放器限制
#### ▫ 支持同时下载 lrc 格式歌词
#### ▫ 支持歌曲封面下载,歌曲属性写入,如歌手、专辑名
#### ▫ 易二次开发:脚本多使用函数构成，方便二次开发
